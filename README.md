# Oregon Fire Casualty Visualization Dashboard
- An interactive dashboard for visualizations comparing Oregon fire casualty data (2020-2024) and housing conditions.
- Casualty data sourced from the Oregon State Fire Marshal Fire Casualties Open Dataset (https://www.oregon.gov/osfm/about-us/pages/geographic-information-system-maps.aspx)
- Housing data sourced from Oregon Housing Needs Analysis (OHNA) (https://data.oregon.gov/dataset/OHNA-Housing-Outcomes-Equity-Indicators-Dataset/ih36-ac7z/about_data)
